## 用户需求

实现 HireInsight-Agent 底层数据清洗模块，包括：

1. 定义原始 Mock 数据的 JSON 结构
2. 定义清洗后输出的 DataFrame 结构
3. 实现数据清洗函数（薪资解析、城市规范化、经验解析、学历规范化等）
4. 实现 Mock 数据生成器
5. 实现 SQLite 数据持久化
6. 编写单元测试

## 功能内容

- 从 JSON 文件加载原始岗位数据
- 使用正则表达式清洗薪资格式（统一为千元/月）
- 规范化城市名称（去重、标准化）
- 解析经验年限要求
- 规范化学历要求
- 将清洗后的数据保存为 DataFrame 并持久化到 SQLite
- 生成数据统计摘要供 LLM 使用

## 视觉效果

无 UI 变更，仅后端数据处理模块

## 技术栈

- Python 3.10+
- Pandas 2.2.3（数据处理）
- SQLite3（内置，数据持久化）
- JSON（Mock 数据存储）
- 正则表达式（数据清洗）

### 2. 核心函数实现策略

#### 阶段一：数据生成（utils/data_generator.py）

- `generate_mock_jobs(count=100) -> List[Dict]`: 生成指定数量的 Mock 岗位数据
- `save_mock_data(filepath: str, data: List[Dict])`: 保存 Mock 数据到 JSON 文件
- `load_mock_data(filepath: str) -> pd.DataFrame`: 从 JSON 加载为 DataFrame

#### 阶段二：数据清洗（utils/data_cleaner.py）

- `parse_salary(salary_str: str) -> Dict`: 解析薪资字符串（增强版，支持更多格式）
- `normalize_city(city_str: str) -> str`: 城市规范化（扩展映射表）
- `parse_experience(exp_str: str) -> Dict`: 经验解析（增强版）
- `normalize_education(edu_str: str) -> str`: 学历规范化（新增）
- `parse_publish_date(date_str: str) -> datetime`: 日期解析（新增）
- `clean_job_data(df: pd.DataFrame) -> pd.DataFrame`: 主清洗函数（增强版）
- `validate_cleaned_data(df: pd.DataFrame) -> Tuple[bool, List[str]]`: 数据校验（新增）

#### 阶段三：数据持久化（utils/data_persistence.py）

- `init_sqlite_db(db_path: str)`: 初始化 SQLite 数据库
- `save_to_sqlite(df: pd.DataFrame, db_path: str, table_name: str)`: 保存到 SQLite
- `load_from_sqlite(db_path: str, table_name: str) -> pd.DataFrame`: 从 SQLite 加载
- `query_jobs(db_path: str, filters: Dict) -> pd.DataFrame`: 条件查询（新增）

#### 阶段四：数据统计（utils/data_cleaner.py 补充）

- `aggregate_job_stats(df: pd.DataFrame) -> Dict`: 统计摘要（增强版）
- `generate_market_summary(df: pd.DataFrame) -> str`: 生成市场摘要文本（新增）

### 3. 性能优化

- 使用 Pandas 向量化操作，避免逐行循环
- 正则表达式预编译（re.compile）
- 城市映射表使用字典 O(1) 查找
- 批量数据库插入（executemany）

### 4. 错误处理

- 数据清洗异常捕获与日志记录
- 无效的薪资/经验格式返回 None 并记录
- SQLite 操作使用事务保证原子性
