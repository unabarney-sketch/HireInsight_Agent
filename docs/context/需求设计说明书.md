AI交互周报：融合版招聘信息实时数据分析系统需求总结


**项目背景与定位变更为混合双端/融合架构**

本项目在陈强《Python项目开发实战》参考书的经典大纲基础上进行重构。系统将在8天开发周期内，利用Python+Streamlit+DeepSeek API技术栈，将传统的“爬虫+数据统计+可视化”与前沿的“AI Agent（智能体）”深度结合。系统表面上满足学校对数据全流程处理的硬性考核指标，核心则通过大模型实现差异化的含金量。

**核心功能需求（融合版 MVP）**

1. **数据获取与极速清洗模块（基础指标）**
    
    - **岗位定向采集：** 采用轻量化方案，通过 Python 抓取大厂招聘官网公开的接口数据或导入高质量本地结构化数据。
        
    - **正则化数据清洗：** 沿用参考书核心逻辑，利用正则表达式对薪资格式进行统一换算（如统一换算为“千元”单位），并规范化城市地区等字段。
        
    - **轻量化存储与统计：** 抛弃传统复杂的 MySQL 数据库配置，采用内置的 SQLite 或直接以 Pandas DataFrame/CSV 形式在本地流转。使用 Pandas 对工作经验、地区、薪资区间和学历水平四个维度进行统计分析。
        
2. **数据可视化大屏模块（教师审核标准）**
    
    - **极速看板渲染：** 舍弃传统的 Flask 路由与前端 HTML/JS 编写，全量转向 Streamlit 框架。
        
    - **多维图表展示：** 直接利用 Streamlit 原生图表或 Pyecharts 组件，将 Pandas 统计好的工资分布、学历占比、工作年限要求等数据以柱状图、饼图等现代高颜值看板实时渲染呈现。
        
3. **AI 智能洞察与诊断 Agent 模块（简历王牌亮点）**
    
    - **大盘市场智能洞察（宏观）：** 部署 Market Agent，自动将 Pandas 的分组统计概要数据喂给 DeepSeek API，利用大模型自身的领域知识自动生成白话文版本的《实时行业风向标与就业形势报告》。
        
    - **简历毒舌匹配诊断（微观）：** 提供 PDF 简历上传解析接口。用户上传简历后，Agent 将简历文本与当前数据库的高频岗位 JD 进行语义级别的交叉对比，实行“找茬”机制（Gap 分析），直接指出用户的技能盲区并动态预测出数道定制化的面试真题。
